Introduction to Information Visualization Riccardo Mazza, Springer Science & Business Media, Feb 28, 2009

If the data visualization filed is totally new to you. This book might be a good option to start with.It gives an overview on different aspects involved in the data visualization process, like perception, multivariate analysis, interactions and how to evaluate visualization works.

Beautiful Visualization: Looking at Data through the Eyes of Experts Julie Steele, Noah Iliinsky, "O'Reilly Media, Inc.", Apr 23, 2010

This book is a collection of different visualization projects from authors with a variety of backgrounds. They are artists, designers, commentators, scientists, analysts, statisticians, and more. This book is great about the beauty of story-telling.

Data Visualization: A Successful Design Process Andy Kirk, Packt Publishing Ltd, Jan 1, 2012

This book is a good guide when you are trying to visualize data yourself.It is also a newbie-friendly.Real examples and illustrative diagrams will help you to better understand the visualization process

Data Visualization For Dummies Mico Yuk, Stephanie Diamond John Wiley & Sons, Jan 6, 2014

This book is a good for "dummies", as indicated in its name. It teaches you how to visualize your data from scratch. Compared with other books recommend here, which are more academic or technical, this book is more business-oriented.

Visualizing Data: Exploring and Explaining Data with the Processing Environment Ben Fry, "O'Reilly Media, Inc.", Dec 18, 2007

This book is more technical. It contains a lot Java code and teach you how to get, refine and visualize data (use the IDE developed by the author). There are a lot of codes in this book, but it also teaches you how to interpret the data.

Network Graph Analysis and Visualization with Gephi Ken Cherven, Packt Publishing Ltd, Sep 24, 2013

Gephi is a popular and powerful software that specializes in analyzing network data with a friendly graphic user-interface.So you can analyze data without coding. This book is a great guide for people who want to learn hand-on skills. It contains a lot of examples.

Tableau Data Visualization Cookbook Ashutosh Nandeshwar, Packt Publishing Ltd, Jan 1, 2013

Tableau is probably one of the most popular data visualization tools on the market today.Although it is a commercial software, it also has public and student versions for people who want to learn this free. This cookbook teaches you step by step how to realize different data visualization using tableau.

Interactive Data Visualization for the Web Scott Murray, "O'Reilly Media, Inc.", Mar 15, 2013

It is an introduction book to people who want to publish interactive data visualization projects on the Web.The approach introduced in this book is D3. It is a javascript library, and also one of the very popular visualization tools today in the market.
