About


This site is a project for the course LIBR 500 Foundations of Information Technology at iSchool@UBC.

This is a responsive site, images and embedded video's are also responsive.

However. for best user experience, it is suggested to view the website use a modern browser and a screen with the resolution of 1366*768. IE 9+ is supported but strongly not recommended.
